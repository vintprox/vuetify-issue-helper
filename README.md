# vuetify-issue-helper

### Project setup
#### Install ⤵️
```
yarn install
```

#### Add .env.local variables
```bash
touch .env.local
echo 'VUE_APP_API_URL=http://url' >> .env.local
```

### Running the application 🔧
#### Compiles and hot-reloads for development
```
yarn run serve
```

### Building 🚧
#### Compiles and minifies for production
```
yarn run build
```

### Deployment 🚀
Install Now CLI, (this will auto-build and does not require `yarn build` prior
This deploys to a temporary URL and DOES NOT prod deploy
```bash
yarn deploy
```

### Testing, etc 🚨
#### Run all tests
```
yarn run test
```

#### Lints and fixes files
```
yarn run lint
```

#### Run your end-to-end tests
```
yarn run test:e2e
```

#### Run your unit tests
```
yarn run test:unit
```
